Methods

private

  • initialize() — Runs the routing
  • parseRoute() — Sets the loaded route parameters
  • redirectAuthMode() — Executes the routing for controllers requiring an authentication
  • redirectFail() — Redirects the browser to a default route
  • redirectFailMain() — Redirects the application to the Application Main Space
  • redirectFailMode() — Redirects the application to the Application Admin Space
  • redirectMain() — Executes the routing for controllers in the Main Application Space
  • redirectTask() — Verifies if a redirection to an API class or Console mode is required
  • setPath() — Sets the path to load
  • setQueryString() — Sets the query parameters
  • setRequestMethod() — Sets the request method
  • setRequireAuthMode() — Requires the authentication mode

  • getDefaultView() — Gets the default view name based on the requested path
  • getFilteredControllerAction() — Gets the controller action name without hyphens
  • getRequestMethod() — Gets the request method
  • isAdmin() — Verifies an Application Admin Space route
  • redirect() — Redirects the browser to a path
